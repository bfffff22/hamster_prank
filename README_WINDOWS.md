# HAMSTER PRANK - Инструкция для Windows

## 🚀 Быстрый старт

### 1. Установка
```bash
# Запусти установщик
install_windows.bat
```

Установщик автоматически:
- Проверит Python
- Установит pip
- Установит все зависимости
- Создаст нужные папки

### 2. Запуск
```bash
# Просто запусти
start.bat
```

## 📋 Требования

- **Windows 7/8/10/11**
- **Python 3.7+** (скачать с https://www.python.org/downloads/)
  - ⚠️ При установке Python поставь галочку "Add Python to PATH"!

## 🎭 Возможности

### Локальные пранки:
1. 🎭 **Матрица** - зеленые символы падают как в фильме
2. 💥 **Глитч** - экран сходит с ума
3. 🌊 **Заливка экрана** - символы летают повсюду
4. 📺 **Полноэкранное окно** - текст на весь экран
5. 🪟 **Управление окнами** - свернуть/развернуть все окна
6. 💣 **Спам программами** - открыть кучу блокнотов

### SSH пранки (для удаленки):
7. 🌐 **SSH: Матрица** - запустить матрицу на удаленной машине
8. 🌐 **SSH: Китайская атака** - полноэкранные китайские символы
9. 🌐 **SSH: Заливка** - залить экран удаленки
10. 🌐 **SSH: Текст** - показать текст на удаленке
11. 🌐 **SSH: Управление окнами** - управлять окнами удаленки
12. 🌐 **SSH: Сессия** - открыть SSH терминал

## 🐹 Анимация хомяка

При запуске программы ты увидишь анимированного хомяка из файла `hom.txt`!

## ⚙️ Настройки

В программе есть меню настроек (пункт 13), где можно:
- Посмотреть сохраненные SSH хосты
- Очистить историю подключений

## 🔧 Решение проблем

### Python не найден
```bash
# Установи Python с официального сайта
https://www.python.org/downloads/

# Не забудь поставить галочку "Add Python to PATH"!
```

### Ошибка при установке зависимостей
```bash
# Обнови pip
python -m pip install --upgrade pip

# Попробуй установить вручную
python -m pip install pexpect pywin32
```

### Не работают пранки
- Убедись что запускаешь от имени администратора (правой кнопкой -> "Запуск от имени администратора")
- Проверь что антивирус не блокирует программу

## 🎯 Примеры использования

### Локальный пранк
1. Запусти `start.bat`
2. Выбери пункт 1 (Матрица)
3. Введи длительность (например 15 секунд)
4. Наслаждайся!

### SSH пранк
1. Запусти `start.bat`
2. Выбери пункт 7 (SSH: Матрица)
3. Введи данные для подключения (user@ip:port)
4. Введи пароль
5. Выбери длительность
6. Пранк запустится на удаленной машине!
