# Hamster Prank - Swiss Army Knife

Швейцарский нож для пранков и удаленного управления через SSH. Работает на Windows и Linux без внешних зависимостей.

## Возможности

### Локальные пранки

**Консольные эффекты:**
- **Заливка экрана** - заполнение консоли случайными символами
- **Матрица** - эффект падающих символов как в фильме
- **Глитч** - эффект искажения экрана
- **Тряска экрана** - быстрая смена содержимого

**Полноэкранные пранки (GUI окна):**
- **Полноэкранный текст** - огромное окно с текстом на весь экран
- **Полноэкранная заливка** - анимированная заливка символами в GUI окне
- **Полноэкранная картинка** - показ изображения на весь экран
- **Спам окнами** - множество полноэкранных окон одновременно

**Текстовые эффекты:**
- Волна, печатная машинка, радуга, приближение, тряска, спам

**Управление системой:**
- Сворачивание/разворачивание всех окон
- Танец окон (циклическое сворачивание)
- Спам программами (блокноты, браузеры)
- Открытие CD привода
- Хаос с громкостью
- Озвучка текста

### SSH пранки
- **Подключение к удаленным хостам** - сохранение истории
- **Заливка экрана на удаленке** - консольные эффекты через SSH
- **Матрица на удаленке** - анимация через SSH
- **Полноэкранный текст** - показ текста на удаленной машине
- **Управление окнами** - сворачивание окон на удаленке
- **Спам программами** - запуск приложений через SSH
- **Выполнение команд** - любые команды на удаленной машине
- **SSH shell** - интерактивная сессия

### Установка софта
- PortProton (Linux)
- Wine (Linux)
- Steam
- Кастомные скрипты

## Модификация

### Добавить свой пранк

Создай файл `pranks/my_prank.py`:

```python
#!/usr/bin/env python3
import sys
from pathlib import Path
sys.path.insert(0, str(Path(__file__).parent.parent))

from core.display import Display

def my_prank():
    display = Display()
    display.clear()
    display.print_centered("МОЙ ПРАНК!")
    # Твой код здесь

if __name__ == "__main__":
    my_prank()
```

Добавь в `main.py` в нужное меню:
```python
elif choice == 'X':
    from pranks.my_prank import my_prank
    my_prank()
```

### Добавить установщик

Создай функцию в `tools/installer.py`:

```python
def install_my_program():

    return True
```

## FAQ

**Q: Нужны ли root/sudo права?**  
A: Нет, большинство функций работают без прав администратора.

**Q: Работает ли без интернета?**  
A: Да, все пранки работают оффлайн. Интернет нужен только для установки софта.

**Q: Как остановить пранк?**  
A: Нажми `q` во время выполнения или `Ctrl+C`.

**Q: Можно ли использовать через SSH?**  
A: Да, программа полностью работает в терминале.
