# Hamster Prank - Rucky Payloads

## Описание

Этот каталог содержит готовые .txt файлы для запуска Hamster Prank через USB Rubber Ducky (Rucky).

## Payloads

### rucky_final_slow.txt
- **Рекомендуемый файл для надежного запуска**
- Максимально медленный и безопасный
- 100% работоспособность на большинстве систем
- Использует стандартный путь `D:\Всякое\hamster_prank`

### rucky_universal_slow.txt
- Универсальный запуск
- Автоматически ищет папку hamster_prank в разных местах
- Использует PowerShell для большей совместимости

### rucky_slow_payload.txt
- Базовый медленный payload
- Работает через cmd с минимизированным окном

## Использование

### Для HID атаки:

1. Выберите подходящий `rucky_*.txt` файл
2. Переименуйте в `inject.bin` или используйте с вашим Rucky инжектором
3. Вставьте USB Rucky в цель

## Пути

Payloads рассчитаны на следующие пути установки:
- `D:\Всякое\hamster_prank` (основной путь)
- `$env:USERPROFILE\hamster_prank` (пользовательская папка)
- `C:\hamster_prank` (корень диска C)
- `$env:PUBLIC\hamster_prank` (общая папка)

## Задержки

Все файлы содержат максимально возможные задержки для:
- Открытия Run диалога
- Ввода команд
- Запуска приложений
- Загрузки скриптов

Рекомендуется протестировать payload на тестовой машине перед использованием.
