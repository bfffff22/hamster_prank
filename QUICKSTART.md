- Спам программами через SSH
- Выполнение любых команд
- Интерактивная SSH сессия

## Запуск:

### Windows:
```
start.bat
```
или
```
python main.py
```

### Linux:
```
chmod +x start.sh
./start.sh
```

## Примеры использования:

### 1. Полноэкранный текст (GUI окно):
```
Запусти программу → 1 → 2 → 1
Введи текст: "ТЫ ВЗЛОМАН!"
Выбери цвета
→ Появится огромное окно на весь экран!
```

### 2. SSH пранк:
```
Запусти программу → 2 → 1
Введи: user@192.168.1.100
Выбери пранк (например матрица)
→ Пранк выполнится на удаленной машине!
```

### 3. Спам окнами:
```
Запусти программу → 1 → 2 → 4
Введи текст и количество окон
→ Множество полноэкранных окон!
```

## Технические детали:

### Полноэкранные окна (Windows):
- Используется PowerShell + Windows.Forms
- Окна создаются через GUI API
- Поддержка настройки цветов
- Закрытие по ESC или Q

### SSH пранки:
- Использует системный SSH клиент
- Выполнение Python кода на удаленке
- Сохранение истории подключений
- Интерактивная сессия

### Без зависимостей:
- Чистый Python 3
- Системные утилиты (ssh, powershell)
- Никаких pip install

## Модификация:

Все модули в папках:
- `core/` - базовые функции
- `pranks/` - пранки
- `tools/` - установщики

Добавляй свои пранки легко!
